# Pre-Launch Checklist

Complete this checklist before making your portfolio public.

## Content Review

### Contact Information
- [ ] Email address updated in `contact.html`
- [ ] Phone number updated in `contact.html`
- [ ] Location information is correct
- [ ] Social media links point to your profiles
- [ ] All social links tested and working

### About Page
- [ ] Bio written and reflects your story
- [ ] Services accurately describe your offerings
- [ ] Pricing/packages mentioned (if applicable)
- [ ] Call-to-action buttons work

### Brand Identity
- [ ] Brand name correct on all pages
- [ ] Tagline/subtitle is accurate
- [ ] Location (Genadendal) mentioned appropriately
- [ ] Footer copyright year is current

## Images & Media

### Portfolio Images
- [ ] Real portfolio images added (not placeholders)
- [ ] Images optimized (see IMAGE-OPTIMIZATION.md)
- [ ] All images under 500 KB
- [ ] Images have descriptive alt text
- [ ] Images properly categorized (weddings, portraits, etc.)
- [ ] Hero image is high quality and representative

### Image Organization
- [ ] Images organized in appropriate folders
- [ ] File names are descriptive (not IMG_1234.jpg)
- [ ] Original high-res images backed up separately
- [ ] Placeholder images removed or replaced

## Technical Setup

### SEO & Meta Tags
- [ ] Page titles are descriptive and unique
- [ ] Meta descriptions written for all pages
- [ ] Open Graph images set (for social sharing)
- [ ] URLs updated from `perfectlyposed.netlify.app` to your domain
- [ ] Canonical URLs point to correct domain
- [ ] Favicon displays correctly

### Functionality Testing
- [ ] All navigation links work
- [ ] Gallery filter buttons work correctly
- [ ] Lightbox opens and closes properly
- [ ] Lightbox keyboard navigation works (arrows, Escape)
- [ ] Contact form configured (Netlify Forms or custom)
- [ ] Form validation works
- [ ] 404 page displays correctly
- [ ] All pages load without errors

### Mobile Testing
- [ ] Site tested on iPhone/iOS
- [ ] Site tested on Android
- [ ] Navigation works on mobile
- [ ] Images display correctly on mobile
- [ ] Forms work on mobile devices
- [ ] Touch interactions work (lightbox, filters)

### Browser Testing
- [ ] Chrome (latest)
- [ ] Firefox (latest)
- [ ] Safari (latest)
- [ ] Edge (latest)
- [ ] No console errors in any browser

## Performance

### Speed & Optimization
- [ ] PageSpeed Insights score 90+ (mobile)
- [ ] PageSpeed Insights score 90+ (desktop)
- [ ] Total page size under 3 MB
- [ ] Images lazy load correctly
- [ ] No render-blocking resources
- [ ] Fonts load efficiently

### Caching & Headers
- [ ] `netlify.toml` configured
- [ ] `_headers` file present
- [ ] `_redirects` file present
- [ ] Security headers enabled
- [ ] Cache headers set for assets

## Deployment

### Netlify Configuration
- [ ] Site deployed to Netlify
- [ ] Custom domain configured (if applicable)
- [ ] HTTPS enabled (automatic with Netlify)
- [ ] Deploy previews working
- [ ] Build logs show no errors

### Domain Setup (if using custom domain)
- [ ] Domain purchased
- [ ] DNS configured correctly
- [ ] Domain propagated (check with DNS checker)
- [ ] www and non-www both work
- [ ] SSL certificate active

### Forms & Integrations
- [ ] Contact form backend configured
- [ ] Form submissions tested
- [ ] Email notifications working
- [ ] Spam protection enabled (if using Netlify Forms)

## Analytics & Tracking (Optional)

- [ ] Google Analytics installed
- [ ] Analytics tracking code tested
- [ ] Conversion goals set up
- [ ] Privacy policy added (if collecting data)

## Legal & Compliance

### Content Rights
- [ ] You own rights to all portfolio images
- [ ] Client permission obtained for published photos
- [ ] Model releases signed (if applicable)
- [ ] No copyrighted content used without permission

### Legal Pages (if required)
- [ ] Privacy policy (if collecting data)
- [ ] Terms of service (if applicable)
- [ ] Cookie notice (if using cookies/analytics)

## Marketing Preparation

### Social Media
- [ ] Social media profiles updated with website link
- [ ] Instagram bio includes website
- [ ] Facebook page updated
- [ ] Pinterest profile linked
- [ ] LinkedIn updated (if applicable)

### Launch Announcement
- [ ] Social media posts prepared
- [ ] Email to existing clients drafted
- [ ] Business cards updated with website
- [ ] Google My Business updated

### SEO Setup
- [ ] Google Search Console verified
- [ ] Sitemap submitted to Google
- [ ] Bing Webmaster Tools configured (optional)
- [ ] Local SEO optimized (Google My Business)

## Final Checks

### Quality Assurance
- [ ] All text proofread for typos
- [ ] All links tested (no broken links)
- [ ] All images display correctly
- [ ] No placeholder text remaining
- [ ] No "lorem ipsum" or dummy content
- [ ] Footer information correct

### User Experience
- [ ] Site loads quickly (< 3 seconds)
- [ ] Navigation is intuitive
- [ ] Call-to-action buttons are clear
- [ ] Contact information easy to find
- [ ] Mobile experience is smooth

### Backup & Security
- [ ] Code backed up to GitHub
- [ ] Original images backed up separately
- [ ] Netlify account secured (2FA enabled)
- [ ] GitHub account secured (2FA enabled)

## Post-Launch Tasks

### Immediate (Day 1)
- [ ] Test site from different locations
- [ ] Share on social media
- [ ] Email announcement to clients
- [ ] Monitor for any issues

### First Week
- [ ] Check analytics daily
- [ ] Monitor form submissions
- [ ] Respond to any inquiries quickly
- [ ] Fix any reported issues

### First Month
- [ ] Review analytics data
- [ ] Optimize based on user behavior
- [ ] Add more portfolio images
- [ ] Collect client testimonials
